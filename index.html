Drawsome

This group project was inspired from "Drawful" by Jackful games and was designed for 3-8 players, each competing on separate web browsers.The project was quite ambitious given a week's duration but was fun and a great team effort.

The game features free hand drawing onto a blank space, made possible through HTML5's Canvas API with help from the Paper.js library. In order to achieve a real-time drawing effect across all clients, websockets were used for client/server communication. While the 'drawer' interacted with the games canvas, graphical data was sent through our Ruby on Rails backend and emitted to all participating players. This was all tied together using a single-page Backbone application to give the user a pleasant gaming experience.

However, we are aware of the problems that can still be resolved and have come up with the following improvements: - Optimise Backbone.js views - Support multiple games running simultaneously - Offer a flexible colour palette - Anti-aliasing on canvas paths.

Plan Easy

Any one planning for an event, from a simple birthday party or a meeting to a large scale wedding, would know how tough and difficult it is to get it organised. PlanEasy is a web application that helps users organise important personal gatherings for special occasions, including parties, graduations and other friend and family focused celebrations. With PlanEasy a user can create an event, manage its guest list, send invites and receive the RSVP status.

The scope of the project is huge and only a part of it was focused hence my future enhancements to this project would include sending personalised invites other than plain text messages, an event planning to do’s tracker, Geo-location related hints

The project was built using Rails4, Mandrill, Bootstrap, and jQuery
