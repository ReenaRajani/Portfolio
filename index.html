Plan Easy

PlanEasy is an event planner that helps users organise any type of event right from birthday parties to conferences. With PlanEasy a user can

  • create an event
  • manage its guest list
  • send invites receive the RSVP status

This is only an MVP and further releases would include features like sending personalised invites, an event to do’s list and tracker, and Geo-location related hints

The project was built using

  • Rails4
  • Postgres as the database
  • Bootstrap for the designing
  • Mandrill for the mailing service
  • jQuery

Drawsome

This group project was inspired from "Drawful" by Jackful games and was designed for 3-8 players, each competing on separate web browsers.

The game features

  • free hand drawing onto a blank space
  • real-time drawing effect across all clients
  • interactive chat session
  • pleasant gaming experience
The project was built using
  • HTML5's Canvas API with help from the Paper.js library
  • websockets for client/server communication
  • Backbone and Rails4
  • Postgres as the database
